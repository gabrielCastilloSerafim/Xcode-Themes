1.  **Clone the Repository**

    Open your terminal and clone this repository.

    ```bash
    git clone [https://github.com/](https://github.com/)[your-username]/[your-repo-name].git
    ```

2.  **Copy the Theme File**

    Copy the `.xccolortheme` file to Xcode's theme directory.

    ```bash
    # First, create the directory if it doesn't exist
    mkdir -p ~/Library/Developer/Xcode/UserData/FontAndColorThemes/

    # Now, copy the theme file
    cp "[Your Theme Name].xccolortheme" ~/Library/Developer/Xcode/UserData/FontAndColorThemes/
    ```

## Applying the Theme in Xcode ðŸš€

1.  **Restart Xcode** to ensure it loads the new theme.
2.  Open Xcode's settings by navigating to **Xcode > Settings...** (or **Preferences...** on older versions).
3.  Go to the **Themes** tab.
4.  Select **[Your Theme Name]** from the list on the left.

That's it! Enjoy your custom theme across all your machines.
